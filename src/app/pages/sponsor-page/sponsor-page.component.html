AltoPonix is a student-run not-for-profit tech research organization developing a new fully automated hydroponics apparatus known as S2. Our planned robotics system is a platform elevator that periodically douses a set of plants in an aqueous nutrient solution while monitoring the chemical composition of the liquid. Because of this, we can grow plants in nearly half the time with more dense foliage resulting in significantly higher yields. We track meaningful and relevant data via a large onboard sensor suite, utilizing software subcomponents that include machine learning state prediction, foliage segmentation, stereo/lidar depth for 3d mapping, and control theory. We also can continuously monitor the parameters of the system with a webapp.
